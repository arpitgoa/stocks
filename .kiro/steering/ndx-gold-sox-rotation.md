## Strategies Compared

### Strategy 1: DCA
- Buy $X/month into NDX, no matter what

### Strategy 3: NDX + Gold % Rotation
Uses NDX/Gold ratio only:
| NDX/Gold Ratio | Rotation | Monthly $ |
|---|---|---|
| >= 9 | Sell 5% NDX → Gold | 100% Gold |
| 7-9 | Sell 2% NDX → Gold | 100% Gold |
| 4-7 | No rotation | 50% NDX + 50% Gold |
| 2.5-4 | Sell 2% Gold → NDX | 100% NDX |
| < 2.5 | Sell 5% Gold → NDX | 100% NDX |

### Strategy 4: NDX + Gold + SOX 3-Asset Rotation
Uses both NDX/Gold and NDX/SOX ratios:

**Core principle:** Gold = safe haven. NDX/Gold decides risk-on vs risk-off. NDX/SOX decides which tech asset to favor — but ONLY during healthy markets (NDX/Gold 4-7).

**Rotation rules:**
| NDX/Gold | Action |
|---|---|
| >= 9 | Sell 5% of ALL tech (both NDX and SOX) → Gold |
| 7-9 | Sell 2% of ALL tech (both NDX and SOX) → Gold |
| 4-7 | Use NDX/SOX for tech rotation (see below) |
| 2.5-4 | Sell 2% Gold → NDX only (ignore SOX) |
| < 2.5 | Sell 5% Gold → NDX only (ignore SOX) |

**NDX ↔ SOX rotation (only when NDX/Gold is 4-7):**
| NDX/SOX | Action |
|---|---|
| >= 6 | Sell 3% NDX → SOX (SOX is cheap) |
| 5-6 | Sell 2% NDX → SOX |
| 3-5 | No rotation |
| 2.5-3 | Sell 2% SOX → NDX (SOX overheated) |
| < 2.5 | Sell 5% SOX → NDX |

**Monthly fresh investment for S4:**
| NDX/Gold | NDX/SOX | Where monthly $ goes |
|---|---|---|
| >= 7 | any | 100% Gold |
| < 4 | any | 100% NDX (never buy SOX in downturns) |
| 4-7 | >= 5 | 50% SOX + 25% NDX + 25% Gold |
| 4-7 | < 3 | 50% NDX + 25% Gold + 25% SOX |
| 4-7 | 3-5 | 33% each |

**Critical rule:** When NDX/Gold < 4 (tech is cheap/crashing), NEVER buy SOX. Always buy NDX. SOX is more volatile and recovers slower.

## Key Findings

1. **Selling NDX rarely helps** — NDX goes up 85% of rolling 12-month periods. Pure sell signals underperform.
2. **Buy-more-during-dips works** — varying buy amount based on signals beats flat DCA.
3. **NDX/Gold ratio catches tops** — flagged Nov 2021 (ratio 9.08) correctly.
4. **NDX/SOX at all-time low (2.13 in Jun 2026)** — extreme SOX leadership = late-cycle signal.
5. **Percentage-based rotation scales** — fixed $ amounts become meaningless as portfolio grows.
6. **Gold is the safe haven, not cash** — rotation to gold preserves value during crashes.
